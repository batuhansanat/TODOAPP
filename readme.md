# **Patika TODO APP Projesi**
Patika PHP ile TODO APP Projesi

## Genel Özellikler

- Üye olma
- Giriş yapma
- Kategori ekleme
- Kategori silme
- Kategori düzenleme
- Kategori listeleme
- Eklenen kategorilere renk tanımlama

Yeni özellikler geliştirildikçe eklenecek.
